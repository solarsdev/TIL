# 시작 서비스의 이해

## 시작 서비스

- 시스템 구동 시 최초로 실행되는 사용자 레벨 프로세스
- 시스템 구동에 필요한 각종 스크립트를 실행 (실행 레벨/모드는 배포판에 따라 차이가 있음)
  - 컴퓨터 이름 설정
  - 타임존 설정
  - fsck로 디스크 상태 확인
  - 파일시스템 마운트
  - /tmp 디렉토리의 오래된 파일 삭제
  - 네트워크 인터페이스 구성
  - 패킷 필터 설정
  - 네트워크 서비스 시작
  - 기타 데몬 시작
